Demostración del modelo de caja

El modelo de caja CSS es esencialmente una caja que envuelve cada elemento HTML. Consta de: bordes, padding, margins, y el contenido.

Esta caja está hecha con el modelo de caja por defecto (box-sizing: content-box). El ancho de 300px se refiere al contenido. A eso hay que sumarle los bordes y el padding 300+(25*2)+(25*2)=400px.
Esta caja está hecha con el modelo de border-box). El ancho de 300px se refiere al ancho total de la caja (300px).
